unsigned  nbrVie = 6 
) throw (std::string)

Constructeur du jeu.

Paramètres:
fichierle nom complet du fichier où puiser le mot, à savoir qu'une ligne du fichier est considérée comme un mot et que le fichier doit se terminer par une ligne vide.
nbrViele nombre de propositions de lettres non présentes dans le mot autorisées. Valeur par défaut : 6.
Exceptions:
std::stringsi le fichier est introuvable ou si on tente de créer une partie avec zéro (0) point de vie.
